# 高级网页排版模版库 🎨

这是一个精心设计的现代网页布局模版集合，旨在展示多种高效、美观且响应式的前端页面排版方式。本项目纯手工打造，不依赖任何第三方框架（如 Bootstrap 或 Tailwind），仅使用原生的 HTML5, CSS3 和 Vanilla JavaScript。

## 🚀 项目亮点

本库集成了五种经典且现代的网页布局模式，涵盖了从沉浸式品牌页到信息流卡片的多种场景：

### 1. 全屏沉浸风 (Hero Layout)
- **特点**：全屏背景图 + 深色半透明遮罩 + 居中核心文字。
- **适用场景**：产品落地页、官网首页、视觉冲击力强的品牌宣传页。

### 2. 左右切割布局 (Split Layout)
- **特点**：50vw 的双栏对分，实现影像与文字的极致平衡。
- **适用场景**：关于我们页面、对比展示页、极简主义产品介绍。

### 3. Z字视线引导布局 (Z-Pattern Layout)
- **特点**：图文交替排版，符合人类自然的阅读习惯 $\rightarrow$ 视觉流转顺畅。
- **适用场景**：功能特性介绍、故事讲述页、详细服务说明。

### 4. 杂志画报网格 (Magazine Grid Layout)
- **特点**：基于 CSS Grid 的不对称错落排版，打破传统栅格的死板，营造高级时尚感。
- **适用场景**：作品集、艺术展演页、数字画报、创意博客。

### 5. 极简独立卡片 (Cards Layout)
- **特点**：响应式网格布局 + 悬停微交互 $\rightarrow$ 信息封装高效且优雅。
- **适用场景**：产品列表、团队成员介绍、资源库索引。

## 🛠️ 技术栈

- **HTML5**: 语义化结构
- **CSS3**: 
  - Flexbox & CSS Grid (核心布局技术)
  - CSS Animations (淡入效果)
  - Media Queries (响应式适配)
  - Backdrop Filter (毛玻璃导航栏)
- **JavaScript**: 原生 ES6+ 用于实现选项卡切换逻辑

## 💻 如何运行

无需任何构建步骤或依赖安装，直接启动即可：

1. 克隆或下载本项目文件夹 `Web_demo`。
2. 使用任意现代浏览器（Chrome, Edge, Firefox, Safari 等）打开 `index.html` 文件。
